飞友网络 值机接口

约定

编码约定

UTF-8字符编码集

接口调用方式

HTTP POST请求方法

参数约定

请求接口前，参数需要进行如下处理

  • 所有中文参数需要通过url encode编译后进行传递。
  • body参数:上一步请求返回数据进行拼接 body拼接
  • 将参数组成为数组，并进行json encode。
  • 本接口中所有方法名区分大小写。

示例：

url = ‘http://URL/checkin/index.php’;
posts = Array(
    'airline' => '3U',
    'useid'   => 'c1aaf7*******1',
    'body'    => 'passengerName=%E9%9F%A9%E7*******%A8%9F&idType=0+%E8%BA%AB%E4%BB%BD%E8%AF%81&idNo=2301051******&mobileNo=18071533938&depCity=XMN', // 中文需要进行url encode
    'action'  => 'FlightList',    // 方法名大小写敏感
);
posts = json_encode(posts);       // 参数进行HTTP post方法发送前需要 json encode
result = http_post(url, posts);

返回结果格式

JSON字符串

关于示例

示例代码皆为伪代码，仅供参考和理解。

  • posts 表示请求时需要传递的参数
  • 'airline' => '3U' 表示数组的一对键值对
  • json_encode() 表示进行json encode
  • http_post() 表示进行HTTP post 请求
  • airline 表示请求的航空公司二字码
  • 接口返回数据为JSON字符串，解析为数组描述
