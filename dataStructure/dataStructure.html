飞友网络 值机接口

数据格式

输出数据结构

struct Result {
  title;      // String 状态 值机成功 | 值机失败
  status;     // String 状态 信息返回时代码所在行号，是一个不确定数值，用于定义错误发生行
  description;// String 描述
  type;       // String 返回消息类型 INFORMATION, FLIGHTLIST, CODE, SEATMAP, ALTER
  form;       // Array  表单数据
  next;       // Next   下一步接口定义
}

表单数据结构

Array form (
  array(
    自定义数据格式1,
    ... ,
    自定义数据格式N
  )
);

下一步接口参数

struct Next {
  action;       // 方法
  header;       // 组合头部信息,主要应对COOKIE
}

form中可能数据结构

座位图数据结构

struct SeatMap {
  name;
  value;
  type = SeatMap
}

航段数据集合

struct FlightList {
  label;
  field = array(); // 二维数组, 行为一条航段，列为表示航段信息的各种数据结构
  type = FlightList
}

  type = Text
}

下拉标签

struct Select {
  label;
  name;
  value = "value:show|";
  type = Select
}

示例：

Array
(
  [label] => 证件类型

  [value] => 0 身份证:身份证|1 票号:电子客票号|2 护照:护照
  [type] => Select
)
