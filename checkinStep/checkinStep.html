飞友网络 值机接口

值机步骤

本节描述各步骤通用操作

调用地址

http://URL/checkin/index.php

具体URL通过合同或由销售人员提供。

调用方式

url = ‘http://URL/checkin/index.php’;
posts = Array(
    'airline' => airline,          // airline 为需要调用的航空公司二字码
    'useid'   => 'c1aaf7*******1', // 我方分配的用户密钥
    'body'    => 'passengerName=%E9%9F%A9%E7*******%A8%9F&idType=0+%E8%BA%AB%E4%BB%BD%E8%AF%81&idNo=2301051******&mobileNo=18071533938&depCity=XMN',  // 传递上一步from中需要填写的内容
    'action'  => action,           // 需要调用的值机操作
);
posts = json_encode(posts);        // 参数进行 json encode
result = http_post(url, posts);     // HTTP post方法调用接口

结果处理

返回结果为JSON 字符串，最外层为 Result 结构

具体数据示例请看各值机步骤

form

Result/form 作为一个数组，存储自定义数据结构。其中Text, Select, Radio 需要反馈给用户填写或者选择.

Next

Result/Next 结构包含下一次的请求方法 action, 和下一次请求需要传入的header数据

body拼接

  1. 将用户填写的数据，按约定处理。
  2. 将处理后的数据以及请求返回的其他数据结构中的name, value键对应的值拼接为 name=value 形式
  3. 再通过 & 连接各个name=value, 作为请求下一步的 body 值。

示例：

'body'=>'passengerName=%E9%9F%A9%E7%A******&idType=0+%E8%BA%AB%E4%BB%BD%E8%AF%81&idNo=23010519******&mobileNo=18071533938&depCity=XMN&depCityCode=XMN&orgId=3UAIR&airlineCode=3U&userID=3U&language=CN'，
