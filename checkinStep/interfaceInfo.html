值机接口信息查询

调用方式

url = ‘http://URL/checkin/index.php’;
posts = []; //参数为空
result = http_post(url, post);

返回结果

{"airline":["3U","CA","CZ","GS","HU","JD","MF","MU","FM","SC","HO","ZH"], "action":["Entrance","FlightList","SelectSeat","Checkin","CancelCheckin"]}

结果解析

array (
  'airline' => 
  array (
    0 => '3U',
    1 => 'CA',
    2 => 'CZ',
    3 => 'GS',
    4 => 'HU',
    5 => 'JD',
    6 => 'MF',
    7 => 'MU',
    8 => 'FM',
    9 => 'SC',
    10 => 'HO',
    11 => 'ZH',
  ),
  'action' => 
  array (
    0 => 'Entrance',
    1 => 'FlightList',
    2 => 'SelectSeat',
    3 => 'Checkin',
    4 => 'CancelCheckin',
  ),
)

结果说明

节点名称 值说明
airline 可值机航空公司代码，可能会添加、删除或临时关闭
action 接口可能会有的操作，后续每步都会有下一个操作节点名称.此处作为可能出现的功能说明使用。如果现有航空公司的值机流程不变，则该节点不会发生任何改变

备注

值机接口信息为静态数据，会根据业务需求增加新航空公司，或修改可调用方法。如果变动，代码发布即会通知。
